Open source / AI infrastructure

Brain Stack

A self-correcting memory layer for LLM agents. Most agents forget everything between sessions, or remember everything and drift on stale facts. Brain Stack is a working pattern for the middle: an agent that recalls what matters, records what it learns and why, and keeps its own memory honest over months of use. The hard part isn't writing things down. It's verification: correction on contradiction, a no-fabrication rule, and a consolidation pass that keeps memory trustworthy.
