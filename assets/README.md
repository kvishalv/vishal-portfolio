# assets

Drop three files here. Until they exist, the site degrades gracefully (the hero shows a "VK" monogram instead of a photo, and the Résumé button will 404).

| File | What | Recommended |
|------|------|-------------|
| `headshot.jpg` | Hero photo | Square, ~480×480, **a clean professional headshot** (not a casual selfie). This is the single highest-impact change on the page. |
| `resume.pdf` | Linked by the "Résumé" button | Export your builder-first résumé to PDF. |
| `og-image.png` | Link-preview image (LinkedIn, X, Slack) | 1200×630. Name + tagline on a clean background. |

No other code references images, so nothing else breaks if these are missing.
